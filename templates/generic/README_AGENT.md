# README_AGENT.md

This repository is prepared for AI coding agents.

Read AGENTS.md first. AGENTS.md is the source of truth.

Before changing files:

1. Inspect the repository.
2. Identify the current task type.
3. For product features, use Spec Driven Development.
4. Do not implement before approved specs.
5. Run ./init.sh before reporting completion.

If your tool supports custom instructions, point them to AGENTS.md.
